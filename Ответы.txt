*******************
Есть ли проблемы в следующем коде?
int i = 1;
object obj = i;
++i;
Console.WriteLine(i);
Console.WriteLine(obj);
Console.WriteLine((short)obj);
-------------
Будет исключение в последней строке – правила упаковки valuetype в object – значение int i сохраняется в куче, obj – указатель на него в стеке. Обратно распаковать сразу в short нельзя – сначала нужно привести к тому типу из которого упаковывали. Т.е. будет работать конструкция Console.WriteLine((short)(int)obj)
-------------



*******************
Есть таблица Users. Столбцы в ней — Id, Name. Написать SQL-запрос, который выведет имена пользователей, начинающиеся на 'A' и встречающиеся в таблице более одного раза, и их количество.
-------------
Select 
count(name) as _count,
name as name
from Test
group by name
having name like 'A%' and count(name)> 1
